## CSS Highlights

- **Sticky navigation bar** with smooth scroll behavior
- **CSS-only animated laptop** in the Education section (3D perspective + `@keyframes open`)
- **Scroll-driven timeline** in the Experience section using `animation-timeline: view()` and `animation-range`
- **Hover micro-interactions** across cards, buttons, nav links, and the profile image
- **Responsive design** with two breakpoints:
  - `≤ 1024px` — stacked hero, column skills, adjusted navigation
  - `≤ 768px` — fully mobile layout with vertical timeline and stacked contact info
